## Installation
```
$ git clone https://github.com/davidacevedo/Contacts-Keeper.git
$ cd Contacts-Keeper
$ npm install
```

## Development
```
$ npm start
```
Runs the project in development mode with hot-reloading of `src` folder.
Open your browser at [http://localhost:3000](http://localhost:3000).
